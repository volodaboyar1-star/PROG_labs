# Data Structures & Algorithms Labs
## 📂 Структура репозиторію

### Lab 2a: Лінійні структури даних
Реалізація класичних списків з використанням шаблонів (`templates`):
* **Static Array List**: Список на основі масиву фіксованого розміру.
* **Vector List**: Реалізація з використанням динамічного масиву (`std::vector`).
* **Linked List**: Однозв’язний список із вказівниками на початок та кінець.

### Lab 2b: Unrolled Linked List
Просунута структура даних (розгорнутий список), що поєднує переваги масивів та зв'язних списків:
* **Unrolled List (Array-based)**: Вузли містять статичні масиви.
* **Unrolled List (Vector-based)**: Вузли містять `std::vector`.
* **Особливість**: Реалізовано механізм каскадного переміщення елементів (cascading shift) при вставці та видаленні для підтримання заповненості вузлів.
